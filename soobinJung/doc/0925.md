> icyHot : Given two temperatures, return true if one is less than 0 and the other is greater than 100.

```java
public boolean icyHot(int temp1, int temp2) {
  int checkMinus = temp1 * temp2;
  if (checkMinus >= 0) {
    return false;
  }
  if ( temp1 > 100 || temp2 > 100 ){
    return true;
  }
  return false;
}
```
